Struct Idx

pub struct Idx<'n, I: Index = usize> { /* private fields */ }
Expand description

lifetime branded index.

§safety

the type’s safety invariant is that all instances of this type are valid indices for [Dim<'n>] and less than or equal to i::signed::max.
