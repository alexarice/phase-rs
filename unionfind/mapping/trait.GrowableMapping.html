fn empty() -> Self

Returns an empty mapping, which can be grown

Source

fn add(&mut self, key: K, value: V) -> Result<(), Self::AddError>

Similar to set, but adds a key to a mapping. Adding may fail due to the implementation being considered full, or the key already being in the set. Note that implementations must return the correct errors to be usable in a union find.

Source

fn len(&self) -> usize

Gets the number of items currently in the mapping.

Source

fn set_or_add(&mut self, key: K, value: V) -> Result<bool, Self::AddError>

Tries to set the value of the element if the element is already in the set (false is returned). If not, the key is added to the mapping and true is returned.

Source
